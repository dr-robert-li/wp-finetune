2. Create `eval/eval_gen.py` (EVAL-01 + EVAL-03):
```python
"""PHPCS pass rate and security pass rate evaluation.

Runs the served model in <wp_gen> mode on held-out test examples,
pipes generated PHP through phpcs, and computes pass/fail rates.
"""
from scripts.dgx_toolbox import get_toolbox
```

The script must:
- Import: `from scripts.dgx_toolbox import get_toolbox; dgx = get_toolbox()`
- Use `openai.OpenAI(base_url=dgx.vllm_endpoint(), api_key="none")` for model calls
- Load test examples from `data/final_dataset/openai_test.jsonl` (filter for wp_gen examples only — those with `<wp_gen>` in user message)
- For each example:
  a. Send the user message to vLLM via OpenAI chat completions API
  b. Extract PHP code from response
  c. Write to temp file, run `subprocess.run(["phpcs", "--standard=WordPress", "--report=json", tmpfile], capture_output=True)`
  d. Parse PHPCS JSON output: `errors == 0` = pass
  e. For security rate: check if phpcs output contains security-related sniff names (WordPress.Security.*)
- Compute:
  - `phpcs_pass_rate = passed / total`
  - `security_pass_rate = security_passed / security_total` (subset where security sniffs apply)
- Save results to `output/eval_gen_results.json`:
  ```json
  {"phpcs_pass_rate": 0.95, "security_pass_rate": 0.98, "total": 500, "passed": 475, "security_total": 200, "security_passed": 196}
  ```
- CLI: `python -m eval.eval_gen [--limit N] [--output PATH]`
- Expose functions for testing: `run_phpcs(code: str) -> dict`, `compute_pass_rate(results: list) -> float`, `classify_security(phpcs_output: dict) -> bool`

3. Create `eval/eval_judge.py` (EVAL-02):
```python
"""Spearman correlation evaluation for judge mode.

Compares model's <wp_judge> scores against PHPCS error counts as ground truth.
"""
from scripts.dgx_toolbox import get_toolbox
```

The script must:
- Import: `from scripts.dgx_toolbox import get_toolbox; dgx = get_toolbox()`
- Use `openai.OpenAI(base_url=dgx.vllm_endpoint(), api_key="none")` for model calls
- Load test examples from `data/final_dataset/openai_test.jsonl` (filter for wp_judge examples — those with `<wp_judge>` in user message)
- For each example:
  a. Extract the code being judged from the user message
  b. Send to model in wp_judge mode, parse `overall_score` from response JSON
  c. Run PHPCS on the same code, count errors
  d. Invert PHPCS errors to score: `phpcs_score = max(0, 100 - phpcs_errors * 5)`
- Compute: `scipy.stats.spearmanr(model_scores, phpcs_scores)`
- Save results to `output/eval_judge_results.json`:
  ```json
  {"spearman_corr": 0.87, "p_value": 0.001, "total_pairs": 500}
  ```
- CLI: `python -m eval.eval_judge [--limit N] [--output PATH]`
- Expose functions for testing: `invert_phpcs_errors(error_count: int) -> float`, `parse_judge_response(response: str) -> dict`
