## Failure Analysis

**REVL-01A (Spearman):** The merged endpoint rho dropped from the pre-merge Tinker-sampler value of 0.294 to 0.240. This is below the D-V4-04 do-no-harm floor of 0.263. The merge degraded the model's calibrated_canonical GT correlation.

**REVL-02 (PHPCS):** 1 of 17 generation samples failed PHPCS (pass rate 0.9412). The 2pp tolerance bar is 0.98 (baseline 1.0 from the merged-v2 model). This is freshly measured on the v4-winner endpoint from Plan 02 eval_gen_results.json, not carried from the dead v4-nolmhead model.

**Confusion Pareto:** Recall on teacher-FAIL is excellent (0.845 > 0.638) — the model correctly catches most code failures. However, false-FAIL rate on teacher-PASS is 0.468, well above the 0.403 P4 Pareto bar. The merged endpoint is over-strict. This aligns with REVL-07 (SOFT): precision 0.628 reflects the same over-strictness pattern.

**Consequence:** `automated_pass=false` → REVL-05 human review does NOT run. Wave 4 promotion is BLOCKED per D-V4-01 (automated-first / human-last).
