Run the deep judge CoT script in bulk mode (no --pilot flag). The script uses checkpoint/resume, so if interrupted it can be restarted safely.

```bash
python scripts/generate_deep_judge_cot.py
```

This generates ~200 examples (default target is 50% of all Phase 1 functions). The script:
- Loads all Phase 1 passed + failed functions
- Samples 3 golden seeds per example as few-shot exemplars
- Applies the quality gate (all 9 dimensions, scores 1-10, verdict PASS/FAIL)
- Checkpoints every 20 examples
- Writes final output to `data/phase4_reasoning/deep_judge_cot/deep_judge_cot_bulk.json`
- Prints summary with total generated, parse failure rate, dimension coverage
