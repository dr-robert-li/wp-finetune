**Seed loading and sampling:**
- `load_seeds()` -- loads both `ugc_seeds.json` and `ugc_boundary_seeds.json`, filters to `seed_type == "deep_judge_cot"`, returns list
- `sample_seeds(seeds, n=3)` -- randomly samples n seeds; weights boundary seeds 2x higher (use `random.choices` with weights based on `defect_subtlety == "boundary"`)

**Few-shot exemplar formatting:**
- `format_seed_as_exemplar(seed)` -- formats a deep_judge_cot seed as text showing code snippet (truncated to 1500 chars), then reasoning: verdict, dimension_analysis (all entries), overall_score, key_observation. Format matches research Pattern 1.

**Source function loading:**
- `load_phase1_functions()` -- scans PASSED_DIR and FAILED_DIR, loads each JSON file, extracts individual functions. Each function becomes a dict with `code`, `source_file`, `source_dir` ("passed" or "failed"), `function_name`. Filters out functions with code length < 50 chars (per research pitfall 6 -- skip empty/minimal functions). Returns list of all functions.

**Citation accuracy verification (addresses review concern #1 -- citation hallucination):**
- `verify_citation_accuracy(result, source_code)` -- scans the `dimension_analysis` values' `analysis` text for any WP_API_CITATIONS strings. For each cited API found in analysis text, checks if that API string also appears in `source_code` OR if the analysis text explicitly states the API is "missing" or "not used" or "should be" or "lacks" (negative citations are valid -- saying "$wpdb->prepare is missing" is grounded reasoning even if the API isn't in the code). Returns a dict with:
  - `total_citations`: int -- total WP API citations found in analysis text
  - `grounded_citations`: int -- citations where the API appears in source_code OR is cited as absent
  - `hallucinated_citations`: list[str] -- API names cited as present but not found in source code
  - `hallucination_ratio`: float -- len(hallucinated) / total if total > 0, else 0.0

**Core generation function:**
- `generate_deep_judge_cot(code, source_info, seeds, client)` -- takes a PHP code string, source metadata, list of CoT seeds, and anthropic client. Samples 3 seeds via `sample_seeds()`. Constructs prompt with:
  1. System instruction: "You are a WordPress code quality assessor producing deep reasoning chains for training data. You MUST analyze ALL 9 dimensions regardless of how many the example seeds show. Seeds only show 2-3 dimensions as examples of analysis depth, not as the complete set."
  2. The 3 formatted seed exemplars
  3. "NOW ANALYZE:" block with the target code (truncated to 3000 chars)
  4. Explicit output schema instruction: "Return JSON with keys: verdict (PASS/FAIL), dimension_analysis (object with ALL 9 dimensions: wpcs_compliance, sql_safety, security, performance, wp_api_usage, code_quality, dependency_integrity, i18n, accessibility -- each having score (1-10) and analysis (string citing specific WordPress APIs by name)), overall_score (0-100), key_observation (string)."
  5. Explicit API citation instruction: "When WordPress APIs appear in the code, name them explicitly: $wpdb->prepare(), wp_verify_nonce(), esc_html(), current_user_can(), check_ajax_referer(), esc_attr(), esc_url(). When WordPress APIs are MISSING from code that needs them, state explicitly what is missing and why. Do not describe behavior abstractly without naming the specific API. IMPORTANT: Only cite APIs that actually appear in the code or that are demonstrably missing. Do not invent citations."

- Calls `call_with_backoff(client, model="claude-sonnet-4-5", max_tokens=2048, messages=[{"role": "user", "content": prompt}])` (NOTE: corrected from `claude-sonnet-4-6-20250514` which doesn't exist; this script path is deprecated per D-13 — use Claude Code agents for bulk)
- Extracts result via `extract_json(resp.content[0].text)`
- Returns the parsed dict or None

**Quality gate per example (strengthened -- addresses review concern #1):**
- `passes_quality_gate(result, source_code=None)` -- checks:
  1. `verdict` is "PASS" or "FAIL"
  2. `dimension_analysis` has all 9 required dimensions
  3. Each dimension has `score` (int 1-10) and `analysis` (non-empty string)
  4. `overall_score` is int 0-100
  5. **NEW -- Citation accuracy check:** If `source_code` is provided, calls `verify_citation_accuracy(result, source_code)`. If `hallucination_ratio > CITATION_HALLUCINATION_THRESHOLD` (0.5), returns False. This means more than half of cited WP APIs must be grounded in the source code or cited as absent. Examples with 0 citations pass (no hallucination possible).
  6. Returns True/False

```python
"""Retroactive citation-accuracy auditor for deep judge CoT artifacts.

Reads a pilot or bulk CoT JSON array, runs verify_citation_accuracy()
against each example's source code, and emits a JSON audit report.

Usage:
    python scripts/audit_citation_accuracy.py <input.json> <output_audit.json>
"""
import json, sys, argparse
from pathlib import Path

PROJECT_ROOT = Path(__file__).resolve().parent.parent
sys.path.insert(0, str(PROJECT_ROOT))

from scripts.generate_deep_judge_cot import verify_citation_accuracy, CITATION_HALLUCINATION_THRESHOLD

def audit_file(input_path: Path, output_path: Path) -> dict:
    examples = json.loads(input_path.read_text())
    per_example = []
    grounded = 0
    total_citations = 0
    total_hallucinated = 0
    examples_with_zero_citations = 0
    for i, ex in enumerate(examples):
        result = ex.get("reasoning", {})
        source = ex.get("code", "") or ex.get("body", "")
        ca = verify_citation_accuracy(result, source)
        per_example.append({
            "index": i,
            "function_name": ex.get("function_name", ""),
            "source_file": ex.get("source_file", ""),
            "total_citations": ca["total_citations"],
            "grounded_citations": ca["grounded_citations"],
            "hallucinated_citations": ca["hallucinated_citations"],
            "hallucination_ratio": ca["hallucination_ratio"],
            "passes_threshold": ca["hallucination_ratio"] < CITATION_HALLUCINATION_THRESHOLD,
        })
        total_citations += ca["total_citations"]
        total_hallucinated += len(ca["hallucinated_citations"])
        if ca["total_citations"] == 0:
            examples_with_zero_citations += 1
        if ca["hallucination_ratio"] < CITATION_HALLUCINATION_THRESHOLD:
            grounded += 1

    citation_validity_rate = grounded / len(examples) if examples else 0.0
    aggregate_hallucination_ratio = total_hallucinated / total_citations if total_citations else 0.0

    report = {
        "input_file": str(input_path),
        "total_examples": len(examples),
        "citation_validity_rate": citation_validity_rate,
        "aggregate_hallucination_ratio": aggregate_hallucination_ratio,
        "total_citations": total_citations,
        "total_hallucinated": total_hallucinated,
        "examples_with_zero_citations": examples_with_zero_citations,
        "threshold": CITATION_HALLUCINATION_THRESHOLD,
        "per_example": per_example,
    }
    output_path.parent.mkdir(parents=True, exist_ok=True)
    output_path.write_text(json.dumps(report, indent=2))
    return report

def main():
    ap = argparse.ArgumentParser()
    ap.add_argument("input", type=Path)
    ap.add_argument("output", type=Path)
    ap.add_argument("--min-validity-rate", type=float, default=0.70,
                    help="Minimum acceptable citation_validity_rate (fraction of examples with hallucination_ratio < 0.5)")
    args = ap.parse_args()
    report = audit_file(args.input, args.output)
    print(f"Audited {report['total_examples']} examples")
    print(f"  citation_validity_rate: {report['citation_validity_rate']:.3f} (threshold: {args.min_validity_rate})")
    print(f"  aggregate_hallucination_ratio: {report['aggregate_hallucination_ratio']:.3f}")
    print(f"  examples_with_zero_citations: {report['examples_with_zero_citations']}")
    if report['citation_validity_rate'] < args.min_validity_rate:
        print(f"FAIL: citation_validity_rate {report['citation_validity_rate']:.3f} < {args.min_validity_rate}", file=sys.stderr)
        sys.exit(1)
    print("PASS")
    return 0

if __name__ == "__main__":
    main()
```
