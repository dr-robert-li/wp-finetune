## Decisions Made

- Citation hallucination threshold set at 0.5 — examples where >50% of cited WP APIs appear only in analysis text but not in source code AND are not cited as absent are rejected
- Negative citations accepted as valid: if analysis says "$wpdb->prepare is missing" when the API is absent from source code, this is grounded reasoning
- Phase 1 functions use the `body` field (not `code`), and functions < 50 characters are filtered as empty/minimal per research pitfall 6
- Boundary seeds weighted 2x in `random.choices` sampling to improve coverage of subtle defect cases in few-shot exemplars
