# Phase 16-01 — Pipeline Lockdown & Repo Cleanup

**Phase:** 16 — Pipeline Lockdown & Repo Cleanup

**Requirements:** PIPE-01, PIPE-02, PIPE-03

## Goal

Freeze the v3.0 method into a repeatable `PIPELINE.md`, deprecate one-off experiment scaffolding into
`deprecated/` without breaking active code, and clean the folder layout so an outside user can clone and
follow along. Push as dr-robert-li.

## Tasks

1. **PIPE-01 — PIPELINE.md.** Document the spine (data -> SFT gen + judge -> merge -> eval) and the three
   conditional compression gates (RL rejected, Sieve full, prune no_winner) with runnable entrypoints,
   gates, and the known Qwen3-30B-A3B result. Keep no-winner gates as conditional re-test stages for the
   next base. [DONE — verified all entrypoints exist on disk]
2. **PIPE-02 — deprecate.** Using the repo map, move clearly one-off drivers (`_04.4_*`, `_p0_*`,
   `_rlev01_*`, seedA/v3/v4 experiment launchers, stale `launch_*.sh` smokes, root stray `judge_batch_18.py`)
   into `deprecated/` with a README. Before each move, grep the active tree for references; do NOT move
   anything an active script/skill imports or calls. Verify no active reference points at a moved file.
3. **PIPE-03 — restructure + root cleanup.** Sweep stray root logs (`judge_batch2*.log`) into `logs/` or
   gitignore; ensure the top level reads cleanly (README, CHANGELOG, JOURNAL, PIPELINE, PROJECT, config).
   Confirm large model dirs and log dirs are handled by `.gitignore` (not moved).
4. Close: SUMMARY, VERIFICATION, STATE + ROADMAP + REQUIREMENTS, final commit + push as dr-robert-li.

## Approach / safety

- Deprecation = move + strip references, never `rm`. Model tombstones stay put (too big; evidence).
- Every move is grep-checked against the active tree first. A moved file with a live reference is a
  regression, not a cleanup.
- Root cleanup prefers `.gitignore` for logs/db over deletion.

- PIPELINE.md covers every stage with entrypoint + gate + known result; no-winner gates preserved. ✅
- Dead drivers in `deprecated/` with README; grep confirms no active reference to a moved file.
