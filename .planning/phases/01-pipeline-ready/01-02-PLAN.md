       **Constants:**
       ```python
       PLUGIN_CSV = Path("/home/robert_li/Desktop/data/wp-finetune-data/wp_top1000_plugins_final.csv")
       THEME_CSV = Path("/home/robert_li/Desktop/data/wp-finetune-data/wp_top100_themes_final.csv")
       REPOS_YAML = Path(__file__).resolve().parent.parent / "config" / "repos.yaml"
       MIN_INSTALLS = 10_000
       MIN_RATING = 80
       MAX_UNPATCHED = 0
       MAX_REPOS = 100
       ```

       **TAG_TO_PATH_FILTERS dict:**
       ```python
       TAG_TO_PATH_FILTERS = {
           "page builder": ["includes/**/*.php", "modules/**/*.php", "widgets/**/*.php"],
           "seo": ["includes/**/*.php", "src/**/*.php"],
           "woocommerce": ["includes/**/*.php", "src/**/*.php"],
           "e-commerce": ["includes/**/*.php", "src/**/*.php"],
           "security": ["includes/**/*.php", "src/**/*.php"],
           "_default": ["**/*.php"],
       }
       STANDARD_SKIP = ["vendor", "node_modules", "tests", "test", "assets", "css", "js"]
       ```

       **filter_row(row: dict) -> bool:**
       - Parse active_installs: `int(str(row.get("active_installs", 0)).replace("+", "").strip() or 0)`
       - Parse rating_pct: `float(row.get("rating_pct", 0) or 0)`
       - Parse unpatched_vulns: `int(row.get("unpatched_vulns", 0) or 0)`
       - Check github_url is non-empty AND starts with "https://github.com/"
       - Return True only if active_installs >= MIN_INSTALLS AND rating_pct >= MIN_RATING AND unpatched_vulns <= MAX_UNPATCHED AND github_url valid

       **assign_quality_tier(row: dict) -> str:**
       - total = int(row.get("total_known_vulns", 0) or 0)
       - unpatched = int(row.get("unpatched_vulns", 0) or 0)
       - rating = float(row.get("rating_pct", 0) or 0)
       - Return "trusted" if unpatched == 0 AND total == 0 AND rating >= 90
       - Else return "assessed"

       **infer_path_filters(tags_str: str) -> list[str]:**
       - Split tags by comma, strip, lowercase
       - For each keyword in TAG_TO_PATH_FILTERS (skip "_default"): if any tag contains keyword, return those filters
       - Fallback: return TAG_TO_PATH_FILTERS["_default"]

       **make_entry(row: dict, repo_type: str) -> dict:**
       - Extract name from slug
       - url from github_url (ensure ends with .git)
       - quality_tier from assign_quality_tier(row)
       - paths from infer_path_filters(row.get("tags", ""))
       - skip_paths = STANDARD_SKIP
       - description = f"{row['name']} — {active_installs:,} installs, {rating_pct}% rating"
       - Return dict with keys: name, url, quality_tier, paths, skip_paths, description

       **convert_csvs_to_repos(plugin_csv=PLUGIN_CSV, theme_csv=THEME_CSV) -> dict:**
       - WordPress Core hardcoded as first entry:
         ```python
         core = [{

             "skip_paths": ["src/wp-includes/js", "src/wp-includes/css"],
             "description": "WordPress Core — auto-passed"
         }]
         ```
       - Read plugin_csv with csv.DictReader, filter with filter_row, make_entry for each
       - Read theme_csv same way
       - If total plugins > MAX_REPOS: sort by active_installs descending, take top MAX_REPOS
       - Return {"core": core, "plugins": plugins, "themes": themes}
