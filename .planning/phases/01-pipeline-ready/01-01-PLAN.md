    1. Create `scripts/utils.py` with these exact functions:

       **extract_json(text: str) -> dict | None** — 4-strategy JSON extraction:
       - Strategy 1: `json.loads(text.strip())`
       - Strategy 2: regex `r'```json\s*([\s\S]+?)\s*```'` then json.loads
       - Strategy 3: regex `r'```\s*([\s\S]+?)\s*```'` then json.loads
       - Strategy 4: regex `r'\{[\s\S]*\}'` (outermost braces) then json.loads
       - Return None if all fail

       **call_with_backoff(client, base_delay=1.0, max_delay=60.0, factor=2.0, max_retries=5, **kwargs) -> anthropic.types.Message** — Exponential backoff:
       - Loop up to max_retries attempts
       - On `anthropic.RateLimitError`: check for `retry_after` attr, use it if present, else use current delay. Add jitter = random.uniform(0, wait * 0.1). Sleep, then double delay (capped at max_delay)
       - On `anthropic.APIStatusError` with status_code >= 500: same backoff
       - On last attempt: raise the exception
       - On success: return the Message object

       **load_checkpoint(phase: str, checkpoint_dir: Path = None) -> dict** — Load or return empty state:
       - Default checkpoint_dir = `Path(__file__).resolve().parent.parent / "checkpoints"`
       - Accept override via parameter (for testing)
       - Return `{"completed": [], "failed": [], "batch_job_ids": [], "timestamp": None}` if no file

       **save_checkpoint(phase: str, state: dict, checkpoint_dir: Path = None) -> None** — Atomic write:
       - mkdir(exist_ok=True) on checkpoint_dir
       - Write to `{phase}_checkpoint.tmp` then rename to `{phase}_checkpoint.json`
       - Set `state["timestamp"]` to `datetime.utcnow().isoformat()`

       **batch_or_direct(item_count: int) -> str** — Route decision:
       - Return "batch" if item_count >= 50 else "direct"

       **make_batch_request(custom_id, system, user_content, model="claude-sonnet-4-6", max_tokens=1024) -> dict** — Format batch request:
       - Return `{"custom_id": custom_id, "params": {"model": model, "max_tokens": max_tokens, "system": system, "messages": [{"role": "user", "content": user_content}]}}`

       **submit_batch(client, requests: list[dict]) -> str** — Submit batch:
       - Call `client.beta.messages.batches.create(requests=requests)`
       - Return `batch.id`

       **poll_batch(client, batch_id: str, poll_interval: int = 60) -> list** — Poll until done:
       - Loop: retrieve batch, check `processing_status == "ended"`, sleep poll_interval
       - Stream results via `client.beta.messages.batches.results(batch_id)`
       - Return list of results

       **parse_batch_results(results: list) -> tuple[list[dict], list[str]]** — Parse results:
       - For each result: if `result.result.type != "succeeded"` -> failures
       - Otherwise extract text from `result.result.message.content[0].text`, run through `extract_json()`
       - If None -> failures. Else add `_custom_id` key and append to successes
