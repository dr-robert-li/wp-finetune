<objective>
Execute the corrective rank×replay grid end-to-end: train all 9 MoE-only candidates on Tinker,
assemble the combined grid manifest, run them through run_grid_eval.py (cheap judge filter →
only-if-pass merge → REVL-04 wp-bench), apply the pre-registered selection rule, confirm the
winner with a second full wp-bench pass, and export the merge-ready winner adapter for Phase 04.4
— OR record a documented escalation if no candidate clears the HARD gate.

Purpose: this is the phase deliverable. The grid measures how far the rank (D-N2 {8,16,32}) and
replay (D-N4 {15,30,50%}) levers push merged codegen toward the 0.4537 baseline before judge
Spearman falls below the v3 floor (0.263). The output is a single merge-ready adapter handed to
04.4's merge-fix re-gate, or a clean escalation.

Output: grid_manifest.json, grid_results.json (with the selection/escalation verdict), and — if a
winner exists — models/tinker_export/wp-reasoning-v4-winner/checkpoint.tar.
</objective>
