## Sampling Rate

- **Readiness gate (Task 1):** one-shot before any training step (gate, blocks on fail).
- **Router-freeze + config echo (dry-run):** one-shot pre-train (static property — single sample sufficient).
- **Gradient norm <3 (first 100 steps):** MLflow per `logging_steps: 5` → over-samples the 100-step window (Nyquist satisfied).
- **Loss divergence:** train loss per `logging_steps`; val loss per `eval_steps`.
- **Parse-failure <5%:** every-other `save_steps` checkpoint — resolution at which the abort can act before wasting a full epoch.
- **OOM at 8192:** pre-train (dry-run + warmup probe) + per-step `MemAvailable` (MemoryWatchdogCallback).
- **Max feedback latency:** one checkpoint interval (parse-check), one logging step (grad/loss).
